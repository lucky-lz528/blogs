Axios是什么

Axios是一个基于promise的HTTP库，简单的讲就是可以发送get、post请求

Axios特性

  • 可以在浏览器中发送XMLHttpRequests
  • 可以在node.js发送http请求
  • 支持Promise API
  • 拦截请求和响应
  • 转换请求数据和响应数据
  • 能够取消请求
  • 自动转换JSON数据
  • 客户端支持保护安全免受XSRF攻击

Axios适用于什么场景

浏览器发送请求，或者Node.js发送请求都可以用到Axios
像Vue、React、Node等项目就可以使用Axios
如果项目中用到了Jquery，那么就不需要用axios，因为Jquery本身就可以发送请求
